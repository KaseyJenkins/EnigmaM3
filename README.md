## Description

Enigma M3 C++17 Emulator

## Requirements
* `C++17`
* `wxWidgets` [3.1.5](https://github.com/wxWidgets/wxWidgets/releases/tag/v3.1.5)

## Features
Implemented:  
* 8 Wheels
* 2 Reflectors
* Plugboard
